# RTL Assignment
## Getting Started

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

For Unit tests:

```bash
npm test
```

For Cypress tests:

```bash
npm run cypress:open
```


Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
